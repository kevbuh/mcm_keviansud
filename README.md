### Problem F: Policy

### Github Commands

Add ALL files that you have changed:
--> git add .

Add certain files
--> git add filename

Commit the changes you have made:
--> git commit -m "write whatever"

Push the changes to everyone:
--> git push

Update your repository
--> git pull

See status of your local changes
--> git status
